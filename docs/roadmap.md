# Roadmap — MesCalories

## ✅ MVP (v1.0) — En cours

- [x] Architecture Clean + Riverpod + GoRouter
- [x] Onboarding 3 slides
- [x] Profil utilisateur complet (âge, sexe, poids, taille, activité, objectif)
- [x] Calcul automatique BMR/TDEE (Mifflin-St Jeor)
- [x] Dashboard journalier avec anneau de progression
- [x] Suivi macros (P/G/L) avec barres de progression
- [x] Saisie texte → analyse IA → résultat nutritionnel
- [x] Saisie vocale → transcription → analyse IA
- [x] Gestion clés API multi-fournisseurs (OpenAI, Gemini, Anthropic)
- [x] Stockage local chiffré (Isar + flutter_secure_storage)
- [x] Historique journalier avec détail des repas
- [x] Thème clair/sombre premium
- [x] Design system complet (couleurs, typo, animations)
- [x] Architecture open source friendly
